# NewSign ✍️

아이들을 위한 사인(서명) 생성 웹앱.

이름을 입력하고 언어(한글/English/中文/日本語)와 스타일을 고르면 나만의 사인을 만들어 줍니다.
결과는 PNG로 저장할 수 있습니다.

## 스타일
특정 유명인의 사인을 복제하지 않고, **스타일 유형**만 표현합니다.
- 기본형 · 귀여운형 · 아이돌st · CEO/리더st · 정치인st · 중세 필기체

## 폰트
언어별 손글씨 폰트는 Google Fonts(OFL 라이선스, 상업 이용 가능)를 사용합니다.

## 배포 (GitHub Pages)
1. Settings → Pages
2. Source: Deploy from a branch → Branch: `main` / `(root)` → Save
3. `https://dkwarrrr.github.io/NewSign/` 로 접속

## 로컬 실행
`index.html`을 브라우저로 열면 됩니다.
