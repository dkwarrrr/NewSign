# NewSign ✍️

이름을 입력하면 손글씨 사인을 만들어 주는 웹앱.

이름과 타입을 고르면 사인이 그려지는 애니메이션과 함께 완성되고, PNG로 저장할 수 있습니다.
서버 없이 브라우저에서 동작합니다.

## 동작 방식
폰트 글자를 그대로 쓰지 않고, 글자의 외곽선을 **골격(중심선)으로 추출**한 뒤
사람이 펜으로 따라 쓴 것처럼 다시 그립니다. 그래서 폰트가 아니라 손으로 그린 사인처럼 보입니다.

입력한 문자에 따라 언어를 자동으로 판별해 어울리는 필기체 폰트를 씁니다.
- 한글 → Nanum Pen Script
- 영문 → Dancing Script (필기체)
- 한자 → Long Cang
- 일본어(가나) → Yuji Syuku

한글은 글자와 글자 사이는 펜을 떼고, 한 글자 안의 자모끼리만 가는 선으로 잇습니다.
영문은 필기체답게 한 줄로 이어 씁니다.

## 타입
- **기본형** — 담백한 손글씨 사인. 마지막 글자에 받침이 있으면 오른쪽으로 꼬리를 빼고,
  받침 없이 모음으로 끝나면 마지막 모음을 아래로 길게 내려긋습니다.
- **아이돌st ⭐** — 기울기와 첫 글자를 강조하고, 이름 아래를 쓸고 나가는 스월(swoosh)을 더합니다.
  이름 옆에는 낙서 꾸밈이 하나 붙습니다 (토끼 · 하트 · 별 · 스마일 · 나선 · 고양이 · 곰돌이).
- **선수st ⚽** — 첫 글자의 초성을 크게 강조하고 뒷글자는 작게 흘려 씁니다.
  이름 옆/아래에는 스포츠 사인 꾸밈이 하나 붙습니다
  (골프 깃발+번호 · 등번호 · 번호+별 · 날짜 · #번호 · 감싸는 원 · 탁구채).

아이돌st와 선수st의 꾸밈은 **"다시 쓰기"를 누를 때마다 무작위로 새로 뽑힙니다.**
(이름을 바꾸면 이름에 따라 고정된 꾸밈으로 돌아갑니다.)

## 조작
- **다시 쓰기 ▶** — 사인을 다시 그립니다 (꾸밈이 있는 타입은 꾸밈도 새로 뽑힘)
- **PNG로 저장 ⬇** — 완성된 사인을 이미지로 저장

같은 이름이면 항상 같은 사인이 나옵니다 (이름 기반 시드).

## 폰트 라이선스
모든 손글씨 폰트는 Google Fonts(OFL, 상업 이용 가능)를 사용합니다.
특정 유명인의 사인을 복제하지 않습니다.

## 배포 (GitHub Pages)

2. Source: Deploy from a branch → Branch: `main` / `(root)` → Save
3. `https://dkwarrrr.github.io/NewSign/` 로 접속
